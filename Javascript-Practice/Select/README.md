# Select

Angular 2 select.

## Project scripts

### Install packages
The only think you need when when you have prerequisites installed is to run:
```
npm install
```

### Custom scripts 

Running in dev mode:
```
npm run dev
```

Running prod build:
```
npm run build
```

Running unit tests:
```
npm test
```

Running e2e tests:
```
npm run e2e
```
